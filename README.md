# LAG CITY - FiveM Server Official Website

Welcome to the official website of **LAG CITY**, a premium FiveM roleplay server. This website is built with React, Vite, and Tailwind CSS, featuring a modern, high-performance design.

## 🚀 Features

- **Modern UI**: Dark-themed, high-contrast design with smooth animations.
- **Responsive**: Fully optimized for mobile, tablet, and desktop.
- **Real-time Stats**: (Simulated) Live player count and server status.
- **Interactive Sections**: Server features, rules, FAQ, and team members.
- **Discord Integration**: Easy access to the community via Discord.

## 🛠️ Tech Stack

- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React

## 📦 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/lag-city-website.git
   cd lag-city-website
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
